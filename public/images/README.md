## Image Specifications

### Hero Images (1920x1080px)

- High-quality professional business/technology themed images
- Should convey innovation, technology, and business success
- Optimal formats: JPG or WebP
- File size: 200-500KB for optimal loading

### Content Images (800x600px)

- Professional business and technology images
- Should align with the content context
- Clear, well-lit, and high resolution
- Formats: JPG or WebP

### Team Photos (400x400px)

- Professional headshot style
- Clean background
- Good lighting and clear facial features
- Square aspect ratio
- Formats: JPG or WebP

### Testimonial Photos (100x100px)

- Professional headshots or avatars
- Round display format will be applied via CSS
- Clear and recognizable faces
- Formats: JPG or WebP

## Recommended Image Sources

For placeholder/stock images, consider:

- Unsplash (unsplash.com) - Free high-quality photos
- Pexels (pexels.com) - Free stock photos
- Pixabay (pixabay.com) - Free images
- Shutterstock (shutterstock.com) - Premium stock photos

## Image Optimization

Before adding images to production:

1. Optimize file sizes using tools like TinyPNG or ImageOptim
2. Use appropriate formats (WebP for modern browsers, JPG fallback)
3. Consider responsive images for different screen sizes
4. Add appropriate alt text for accessibility
